김미선 / 이력서

안녕하십니까. PG사에서 1차 PG 프로젝트에서 신용카드 승인 파트를 담당하였습니다.
자사 인터페이스에 맞추어 사용자에게 결제 환경을 제공하고, API 규격과 고객 요구사항을 적절히 반영한 프로세스를 구현 테스트를 거듭하여 발전시켜나가는 개발자로 근무하였습니다.

기술 스택

  • 언어 (자바, 자바스크립트)
  • API 테스트 툴(Insomnia)
  • CI/CD(GitHub, Jenkins)
  • 프로젝트 관리 툴(Notion, Jira, Confluence)

현재 00사의 신입 QA 엔지니어로 지원하게 된 김미선입니다.

자사 서비스 테스트 이력

  • WEB/MOBILE
    • 프로젝트 통합 테스트 체크리스트 작성
    • 테스트 시나리오 작성
    • 테스트 진행
    • 개발 건 테스트 케이스 작성
  • APP
    • 안드로이드 단말기 앱 블랙박스 테스팅(테스트 케이스)

위의 테스트 과정에 참여하여 결함 검출 및 이슈 해결을 통해 서비스 품질 개선에 기여하였습니다. 이러한 경험으로 프로젝트의 안정성 및 신뢰성을 높여 검증된 서비스를 배포하는 QA 활동의 전문성을 키워 개발자 관점에서 진행했던 테스팅 범위를 확장하고자 커리어 방향을 전환하게 되었습니다.

직전 근무처에는 QA 전문 조직이 없었던 관계로 이직을 준비하게 되었습니다.

금년 시행하는 CSTS 일반등급(FL) 2차 시험에 응시 하였으며 이후 CSTS AL 또는 ISTQB 라이센스를 취득하여 QA 엔지니어로서 전문성을 갖추는 것을 목표로 하고 있습니다.

개발자 경력으로 쌓인 프로그래밍 실무 역량 및 테스트 경험은 업무 대상 파악 및 테스트 설계에 이점이 될 것이며, 협력 부서와의 원활한 커뮤니케이션 능력 및 사용자/프로세스 기술 문서 작성에 강점이 있다는 업무 평가를 받은 점을 바탕으로 주어진 업무를 수행하여 업무 성과에 힘쓰겠습니다.

사용자에게 기획 의도대로 준비된 서비스를 제공 할 수 있도록 00사의 QA 팀원으로 합류하여 업무에 함께 하기를 희망합니다.
